# Max Vault

A personal thinking hub for planning, decisions, actions, projects, and daily execution.

## Purpose
- Capture and organize thinking in one place.
- Link ideas to people, organizations, clients, and projects.
- Keep an actionable daily workflow.

## Structure
- `00_System/` — workflows, indexes, operating notes
- `01_People/` to `04_Projects/` — core entities
- `05_Content/` — reusable content and drafts
- `06_Interactions/` — interaction logs
- `07_Daily/` — daily notes
- `99_Templates/` — note templates

## Git Notes
This repository intentionally ignores local/editor state, especially machine-specific Obsidian files (for example workspace and cache files), so commits stay focused on durable vault content.

## Daily Use
1. Create or update notes in the relevant folder.
2. Link notes from indexes and project pages.
3. Commit meaningful changes with clear messages.
