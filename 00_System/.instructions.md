This folder contains the operating rules and system state for Max OS.

- `LLM Operating Manual.md` is the primary AI instruction set — read it first in any session.
- `System State.md` has canonical date checkpoints — never overwrite `last_*_review_date` fields without completing the corresponding review.
- `Planning Cadence.md` defines when reviews are due.
- `Planning Memory.md` captures lessons learned from past planning cycles.
- `Indexes.md` maps where each type of note belongs.
- Do not create new system files here unless explicitly requested.
