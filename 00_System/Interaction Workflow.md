# Interaction Workflow

## Goal
Capture each conversation once, then update all linked notes.

## Steps (2-3 minutes)
1. Create a new note in `06_Interactions` using `TPL - Interaction`.
2. Fill in people, org/client/project links.
3. Paste rough notes from the call/chat.
4. Ask Copilot to run the update prompt in [[00_System/Copilot Update Prompt]].
5. Review changes Copilot made in linked person/client/project/org notes.

## Naming Convention
- Interaction note title format:
  - `YYYY-MM-DD - Person A - Topic`
  - Example: `2026-02-24 - Anna Meyer - Q2 Proposal`

## Minimum Required Fields
- `people`
- one of `organization`, `client`, or `project`
- `outcome`
- at least one `Action Item`
