# Copilot Update Prompt

Use this after you finish an interaction note.

For session startup and planning cadence checks, use [[00_System/Session Start Protocol]].

---

I just added/updated this interaction note: [[PASTE INTERACTION NOTE LINK]].

Please do the following in my vault:
1. Ensure every person mentioned has a note in `01_People` (create from `TPL - Person` if missing).
2. Ensure linked organization/client/project notes exist (create from templates if missing).
3. Update each relevant person note with:
   - latest interaction date
   - key facts learned
   - open loops and next actions
   - links to organization/client/project
4. Update the linked organization/client/project note with key updates and commitments.
5. Keep edits minimal and factual; no invented details.
6. Give me a short summary of what you changed.
