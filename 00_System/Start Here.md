## Operating Rules
1. Every meaningful conversation gets an interaction note in `06_Interactions`.
2. Every interaction note links people + org/client/project.
3. Person notes are the source of truth for relationship context.
4. Project notes track delivery and commitments.
5. Client notes track account-level status and risk.
