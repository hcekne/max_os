# System

This folder is mainly for AI assistants.

## Purpose
Store the operating rules and current state that AI uses to help the user consistently.

## Main files
- `LLM Operating Manual.md` - primary AI instructions.
- `System State.md` - latest review dates and active plan links.
- `Planning Cadence.md` - timing rules for weekly/monthly/quarterly/yearly reviews.
- `Planning Memory.md` - lessons to improve future planning.
- `Indexes.md` - where each type of note belongs.

## Human guidance
Most users do not need to edit this folder often.
Use `README.md` in the vault root as your main guide.
