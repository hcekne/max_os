# Inbox Instructions

Raw captures land here. Process daily.

- Follow the Inbox Processing Algorithm in `00_System/LLM Operating Manual.md`.
- Route each item to: people, organization, client, project, interaction, content, or todo.
- Create missing notes from templates in `99_Templates/`.
- Extract explicit tasks to `08_Todos/` or today's daily note in `07_Daily/`.
- Add cross-links between all touched notes.
- After routing, move every processed item out of the active root of `10_Inbox/`.
- Use `10_Inbox/Processed/` as the default destination for processed raw captures unless a more specific processed subfolder applies.
- Do not leave already-routed article drafts, transcripts, or processed captures beside active inbox items.
- When scanning inbox folders, use direct directory listings as the source of truth before relying on globbed search.

## PDF Profiles
- `PDF_Profiles/Unprocessed/` — drop LinkedIn profile PDFs here.
- Process with the skill: `15_Skills/Skill - Process PDF Profiles to People Notes.md`.
- See `14_Guides/Guide - Export LinkedIn Profiles as PDF.md` for export instructions.
- Before reporting that no PDFs were found, list `PDF_Profiles/Unprocessed/` directly.
- After successful processing, move source PDFs to `PDF_Profiles/Processed/` and extracted text to `PDF_Profiles/Extracted_Text/`.
