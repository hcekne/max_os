Agent-executable skill definitions live here. Each skill describes a task an AI agent can perform autonomously.

- Filename pattern: `Skill - Name.md`.
- Each skill has: Purpose, Trigger, Inputs, Steps, Outputs, Quality Checks.
- When creating a new skill, use the template from `99_Templates/TPL - Skill.md`.
- The master manifest of all skills is `SKILLS.md` at the vault root.
- For human-led step-by-step processes, see `12_Workflows/`.
