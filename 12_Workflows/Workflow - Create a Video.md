# Workflow - Create a Video

## Goal
Move from idea to published video with a repeatable process.

## Inputs
- topic and target audience
- platform (YouTube, LinkedIn, etc.)
- desired video length

## Steps
1. Define one-line video promise.
2. Draft simple structure (hook -> points -> close).
3. Create speaking outline or script.
4. Record first take.
5. Edit and add captions/graphics.
6. Write title, description, and thumbnail idea.
7. Publish and log performance baseline.

## Outputs
- published video link
- repurposing notes (clip ideas, post ideas)

## Quality checklist
- [ ] Hook is clear in first seconds
- [ ] Message is easy to follow
- [ ] Audio is clean
- [ ] Close includes clear next step

## Automation opportunities
- Ask AI for hook options and outline drafts.
- Ask AI to create repurposing snippets from the script.
