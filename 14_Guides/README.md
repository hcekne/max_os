# Guides

Use these guides to connect Max OS with your preferred AI assistant.

## Choose one

- Codex CLI: `Guide - Codex CLI.md`
- Claude Code: `Guide - Claude Code.md`

All guides do the same thing:
1. Open your Max OS folder.
2. Start an AI session in that folder.
3. Give the AI a startup prompt so it reads Max OS system files first.
