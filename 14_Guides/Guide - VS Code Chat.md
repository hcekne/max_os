# Guide - VS Code Chat

## What this does
Run Max OS using the chat panel inside VS Code.

## Steps
1. Open VS Code.
2. Open your Max OS folder.
3. Open Chat in VS Code.
4. Paste this startup prompt:

"Use this workspace as Max OS. Read `00_System/LLM Operating Manual.md`. Check `00_System/System State.md` and tell me what I should do today. Then process anything in `10_Inbox/` and give me my top 3 actions."

## Daily use prompts
- "Process pending items in `10_Inbox/`."
- "Create today’s daily note and draft my plan."
- "Update `00_System/System State.md` based on what we completed."
